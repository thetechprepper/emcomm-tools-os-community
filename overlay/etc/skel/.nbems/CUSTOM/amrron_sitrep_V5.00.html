AmRRON Situation Report (SitRep)
v5.00 Use this form to report or update a situation or event in a specified location or region based on personal or reliable sources.
Sample file naming protocol: "CALL-ST-RR-YYMMDD-HHMMZ-REPORT"

1. To: (Recipient)
 2. From: (Sender)
 		3. Message Precedence:
4. State (ST): (State)
 		5. Grid: (Of reported location)
 		6. Size and Scope:
7. DTG: (YYMMDD-HHMMz)
 		8. Expiration: (YYMMDD-HHMMz)
 		9. Status: (Incident Status)
10. Narrative: (Concisely describe the situation)
End of Report

Instructions

  1. TO (Recipient): Who or what group is being addressed.
  2. FROM (Sender): Your FCC Call Sign.
  3. Message Precedence: Select option which best reflects priority or urgency.
  4. State: 2 letter abbreviation of the State for which the StatRep is referring. (ST)
  5. Grid: Maiden Head grid of location being reported. If a more specific location is required, note it in section 11 "Narrative".
  6. Size and Scope: Select “Local,” “Regional,” , “National,” or “Unknown” to reflect size and scope of the event.
  7. DTG: YYMMDD-HHMMz.
  8. Expiraton DTG: YYMMDD-HHMMz.
  9. Status: Select “New,” “Ongoing,” , “Resolved,” or “Unknown” to reflect the status of the event.
  10. Narative: This is the "meat and potatoes" of the SitRep. Describe in the shortest most concise report possible.
