NOAA Weather Radio (NWR) provides continuous weather information,
alerts, and emergency broadcasts across the United States.


## Frequencies

The following are the standard NOAA Weather Radio channels in the
VHF band. These are fixed nationwide frequencies.

| Channel | Frequency (MHz) |
|---------|----------------|
| WX1     | 162.400        |
| WX2     | 162.425        |
| WX3     | 162.450        |
| WX4     | 162.475        |
| WX5     | 162.500        |
| WX6     | 162.525        |
| WX7     | 162.550        |


**Notes:**

- All channels use **FM (narrowband)** modulation
- Coverage depends on local transmitter locations
